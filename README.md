# 项目简介

Python基于YOLOv8+pyqt5的驾驶员行为检测系统是一款专注于驾驶员行为监测的软件。该系统旨在通过先进的计算机视觉技术，对驾驶员在驾驶过程中的各种行为进行实时检测和分析。

## 核心功能
- **数据集支持**：系统内置9884张驾驶员行为数据集，涵盖5类行为（Open Eye, Closed Eye, Cigarette, Phone, Seatbelt）。
- **模型替换**：用户可自行替换模型，以适应不同的检测需求。
- **数据集赠送**：提供3547张图片及相应的xml和txt标签数据集，适用于YOLOv8模型训练。
- **可视化界面**：基于pyqt5开发，提供直观的用户界面，方便用户进行操作和监测。

## 适用场景
- **安全驾驶监测**：在驾驶培训、长途运输等领域，用于监测驾驶员的驾驶行为，保障行车安全。
- **智能交通管理**：辅助交通管理部门对驾驶员的违规行为进行监控，提升道路安全水平。
- **自动驾驶研发**：为自动驾驶技术研发提供数据支持，帮助系统更好地理解驾驶员行为。

## 技术特点
- **YOLOv8算法**：采用先进的YOLOv8目标检测算法，实现对驾驶员行为的快速、准确识别。
- **pyqt5界面**：采用pyqt5构建用户界面，提供良好的交互体验。
- **开放性**：系统代码可运行，方便用户根据需求进行二次开发。
